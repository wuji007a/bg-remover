## 🔑 Database ID 是什么？

**格式：** 36 个字符，用 `-` 分成 4 组

**示例：**
```
72eb7500-430f-21d1-be48-258cb32515c
```

**作用：** 用于连接到你的 D1 数据库

**重要：** 每个 Database ID 都是唯一的，你需要复制你自己创建的那个

## ⚠️ 常见问题

### Q1：我应该选哪个指南？
A: 如果你第一次用，选网页详细指南。如果熟悉了，选快速指南。

### Q2：创建失败怎么办？
A: 检查数据库名称是否是 `bg-remover-db`，位置是否选择正确。

### Q3：SQL 脚本在哪里？
A: 在 `bg-remover` 项目文件夹中，文件名是 `init-d1.sql`。

### Q4：执行 SQL 失败？
A: 确保你复制了完整的 SQL 脚本，重新粘贴并执行。

### Q5：如何知道创建成功？
A: 看到页面显示 "Successfully created" 和 "Successfully executed 6 queries"。

### Q6：Database ID 在哪里？
A: 在数据库详情页面，通常在页面顶部或左侧。
